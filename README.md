# commandTool README
 这是一个可以快速创建项目命令菜单的拓展，通过按钮点击即可运行项目命令。

## Features

- 用文件夹的形式去展现项目命令
- 可以在项目自定义命令
- 可以设定vscode的全局工作空间命令
